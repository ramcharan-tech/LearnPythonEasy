# Blind 75 Problems

## Overview
The Blind 75 is a curated list of 75 essential coding problems that are frequently asked in technical interviews. This repository contains Python solutions to these problems, helping you to prepare effectively for coding interviews.

## Project Structure
```
Blind75
├── problems
│   ├── 01_two_sum.py
│   └── __init__.py
├── README.md
```

## How to Run the Solutions
1. Ensure you have Python installed on your machine.
2. Navigate to the `Blind75` directory in your terminal.
3. Run the desired problem file using the command:
   ```
   python problems/01_two_sum.py
   ```
